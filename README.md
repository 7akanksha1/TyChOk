**TyChOk – Pet Products E-Commerce Website**
TyChOk is a fully-authorized e-commerce website designed exclusively for pet products.
It provides a smooth shopping experience with secure user authentication and a dynamic backend built with PHP.

**Features**
E-Commerce Features
Browse pet products easily
Product details page
Add to cart
Order placement system
User-friendly UI for pet lovers

**Authentication**
Secure user registration
Login & logout system
Password protection
Session-based user validation

**Frontend**
Fully responsive design
Built using HTML, CSS, JavaScript
Clean and intuitive layout

**Backend**
PHP-based backend
Database operations using MySQL(via.XAMPP)
Validations for all user inputs
Admin-friendly architecture (optional)

**Tech Stack**
Frontend	HTML, CSS, JavaScript
Backend	PHP
Database	MySQL(via.XAMPP)
Server	XAMPP (Apache + MariaDB)

**Installation & Setup**
1️⃣ Clone the repository
git clone https://github.com/7akanksha1/TyChOk.git

2️⃣ Move project to XAMPP htdocs

Place the project folder inside:
C:/xampp/htdocs/

3️⃣ Start XAMPP
Start Apache
Start MySQL

4️⃣ Import the database
Open phpMyAdmin
Create a database (example: tychok)
Import the .sql file included in the project (if available)

5️⃣ Open the project in browser
http://localhost/TyChOk/
